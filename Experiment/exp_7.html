² 知识点

n Html标记选择符

当需要对同一个标记下的所有内容设计同样的样式时，选择符可以它

div {

height: 100px;

width: 200px;

}

n 类

可以对需要设计为同一样式的内容添加一个类，然后对类设计样式

.d1 {

background-color: #FC6;

}

n Id

需要对一个单独区域设计样式时，可以为它添加id，再对此id设计样式

#xx {

border: 5px groove #00F;

}
