n 实验要求

本实验css样式表不添加类和id，全部使用复合选择器实现。（使用复合选择器的好处是无需定义太多的类和id，比较容易找到要定义的标记）

1、打开网页时让下拉菜单隐藏起来（从图3看到，没有设置前下拉菜单显示出来了，看起来页面非常乱），另保存为学号-1.html。

2、鼠标移动到菜单栏时，菜单栏背景色改为rgb(43,95,152)，完成这步后浏览器看到的效果如图4所示。另保存为学号-2.html

 

图3 第2步完成后效果图

3、第2步完成后菜单栏还有2处不满意的地方，（1）菜单栏太宽；（2）菜单栏高度比nav高度小（可以对照信息学院网站，对比效果）

修改要求：（1）调整菜单栏的宽度为10%（调整后菜单栏宽度变小了，但是菜单之间的距离变窄了）；（2）请自己想办法再修改一个属性，使得它们的距离还和以前一样；（3）修改菜单栏的高和nav高一样（nav设计了padding-top，可以删除。这一步就是设计菜单栏在nav中垂直居中，参考行级元素垂直居中），另保存为学号-3.html。

 

图3 第3步完成后效果图

4、当鼠标移动到菜单栏上时，下拉菜单出现。（无需增加类和id，使用E：hover选择器，E部分使用复合标记即可，hover后也可使用标记）如图4所示。另保存为学号-4.html

 

图4 下拉菜单未调整时效果

5、从图4看到下拉菜单还有许多地方需要进行修改，特别是下拉菜单宽度不够，看到的文字都是竖着显示。重点需要修改以下几个方面：

（1）学号-4.html中ul标记样式有部分属性只适用于第一级ul标记，因此要将部分样式复制到第1级ul标记（提示：使用复合选择器的子代选择器nav>ul）（2）下拉菜单的宽度设为140px （3）第一级li标记定位为相对定位（4）第二级ul标记定位为绝对定位。

修改后的样式表，其中Body、nav标记样式不变，其他样式如下

第一级ul标记（提示nav>ul）

{

在nav层中水平方向居中

高60px

其中的文字水平居中

}

第一层ul的li标记 {

宽为10%

步骤3第（2）步设计的属性

左浮动

位置定位为相对定位（下拉菜单定位为绝对定位，上层容易必须定位为相对定位）

}

第2层ul标记{

display属性为none

高度为自动

宽度为140px

定位为绝对定位

背景颜色为rgb(34,79,134);

   左填充为0像素

}

当鼠标移到到第一级li标记后{

背景颜色改为rgb(43,95,152);

}

当鼠标移到到第一级li标记后ul标记{

display属性为block（让下拉菜单显示）

}

第二级li标记{

文本水平方向左对齐

宽140px；

左边-15px（从图4可以看到下拉菜单内容没有对齐菜单栏，因此向左移动一定像素）

背景颜色为rgb(34,79,134);

加框（颜色为#CCC，线条为细实线）（使用border属性）

框上、左、右部线条无（使用border-top-style等属性）

}

另存为学号-5.html

6、为下拉菜单项“专业介绍”添加链接，链接地址为：http://cs.hzu.edu.cn/zyjs/list.htm，链接后弹出新窗口显示该网页内容。

对链接设计以下样式：

（1）链接文字不显示下划线

（2）链接文字颜色为白色

（3）鼠标移到链接文字后，文字变为绿色（提示：设计a:hover样式，但它必须放在a:link和a:visited样式后才生效）

另存为学号-6.html
